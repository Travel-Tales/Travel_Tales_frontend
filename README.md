## 주요기능

- 로그인(카카오, 구글)
- 로그아웃
- 여행 계획, 리뷰 작성페이지
- 소켓 연결 (실시간 공유 가능)
- 여행 계획 작성시 마크다운 포멧 제공
- 여행 계획, 리뷰 리스트 페이지 (카테고리 분류)
- 사용자 프로필 수정

### 설치

Front-end

```
$ git clone 'https://github.com/Travel-Tales/Travel_Tales_frontend.git'
// nvm 권장
$ nvm use 18
$ npm install
```

```
$ npm run dev
```

Back-end

```
$ git clone 'https://github.com/Travel-Tales/Travel_Tales_backend.git'
// nvm 권장
$ nvm use 18
$ npm ci
```

```
$ npm run start:dev
```
